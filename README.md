这是游戏坎巴拉太空计划中的一个mod，它还原了我们太阳系中的一颗短周期彗星 46P Wirtanen。
我建议你在真实太阳系mod（RealSolarSystem）中使用它，而不是在原版使用，因为这会使得它的近日点在Eeloo之外。
我会在后续中加入对Principia的支持。 
但遗憾的是我们目前无法为彗星添加彗尾。
